## 🛠️ O que estou Estudando na Escola?

No Ensino Fundamental 2, a tecnologia é nossa grande aliada. Estes são os softwares e blocos lógicos que estamos dominando em sala de aula para criar os primeiros projetos:

* **Scratch:** Programação lútica e lógica visual baseada em blocos para dar vida a histórias e jogos.
* **JavaScript:** A linguagem que faz a mágica acontecer e dá inteligência aos sites.
* **GitHub:** Nossa pasta virtual onde guardamos e compartilhamos nossos códigos com o mundo.
